Щоб зарелізити готовий проєкт на GitHub, вам потрібно виконати кілька кроків. Ось покрокова інструкція:

1. **Створення нового репозиторію на GitHub:**
   - Увійдіть у свій обліковий запис GitHub.
   - Натисніть на кнопку "New" у верхньому правому куті, щоб створити новий репозиторій.
   - Введіть назву репозиторію та, за бажанням, опис. Виберіть, чи буде репозиторій публічним чи приватним.
   - Натисніть "Create repository".

2. **Ініціалізація локального репозиторію:**
   - Відкрийте термінал (або командний рядок) на вашому комп'ютері.
   - Перейдіть до папки з вашим проєктом за допомогою команди `cd /шлях/до/вашої/папки`.
   - Ініціалізуйте локальний репозиторій командою:
     ```bash
     git init
     ```

3. **Додавання файлів до репозиторію:**
   - Додайте файли до індексу Git:
     ```bash
     git add .
     ```
   - Якщо ви хочете вибрати конкретні файли для завантаження, замініть `.` на назви файлів або шлях до них.

4. **Коміт змін:**
   - Зробіть коміт змін з повідомленням:
     ```bash
     git commit -m "Initial commit"
     ```

5. **Додавання віддаленого репозиторію:**
   - Поверніться до сторінки вашого репозиторію на GitHub.
   - Скопіюйте URL вашого репозиторію (HTTPS або SSH).
   - Додайте віддалений репозиторій до вашого локального репозиторію:
     ```bash
     git remote add origin https://github.com/ваше_ім'я_користувача/назва_репозиторію.git
     ```

6. **Вивантаження файлів на GitHub:**
   - Вивантажте ваші зміни на GitHub:
     ```bash
     git push -u origin master
     ```
   - Якщо ви використовуєте іншу гілку, замініть `master` на назву вашої гілки.

Після виконання цих кроків ваш проєкт буде завантажено на GitHub. Ви можете переглянути його на сторінці вашого репозиторію.
